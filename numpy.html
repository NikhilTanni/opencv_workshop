NumPy

NumPy is a Python package. It stands for 'Numerical Python'.
It is a library consisting of multidimensional array objects and a collection of routines for processing of array. Numeric, the ancestor of NumPy, was developed by Jim Hugunin. Another package Numarray was also developed, having some additional functionalities. In 2005, Travis Oliphant created NumPy package by incorporating the features of Numarray into Numeric package. There are many contributors to this open source project.

Codes and snippets
  • import statement
    import numpy as np

  • Example 1
    
import numpy as np
a=np.array([1,2,3])
print(a)
    Output : [1,2,3]
  • Example 2
    
# more than one dimensions
import numpy as np
a = np.array([[1, 2], [3, 4]])
print(a)
    Output : 
    [[1, 2]
[3, 4]]
  • Example 3
    
# dtype parameter
import numpy as np
a = np.array([1, 2, 3], dtype=complex)
print(a)
    Output : 
    [ 1.+0.j, 2.+0.j, 3.+0.j]
  • Example 4
    
# zeros - creates np array of specified dimension (x,y) and fill with zero
a=np.zeros((x, y))
b=np.zeros((2, 3))
print(b)
    Output : 
    [[ 0., 0., 0.], [ 0., 0., 0.]]
  • Example 5
    
# ones - creates np array of specified dimension (x,y) and fill with one
# a=np.ones((x, y))
b=np.ones((2, 3))
print(b)
    Output : 
    [[ 1., 1., 1.], [ 1., 1., 1.]]
  • Example 6
    
# shape - returns the dimensions of array
b=np.ones((2, 3))
b.shape
    Output : 
    (2,3)
  • Example 7
    
# full - creates np array of mentioned x,y and fill with constant as z
# np.fill((x,y), 7)
# dtype is optional
a=np.full((3, 5), 7, dtype=int)
print(a)
    Output : 
    [[7, 7, 7, 7, 7],
[7, 7, 7, 7, 7],
[7, 7, 7, 7, 7]]
