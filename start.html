Quick Start

Last updated: Nov 21st, 2018

Download

we will be using these tools (or plugings), please install as is in list!

  • Python (version: 3.6.X)
    • PIP/PIP3
    • MatPlotLib
  • Open CV
  • NumPy

Download Python

Installation

Step One

Install Python
follow instructions and install python 3.6.X
Note : make sure you check/tick "Add to path" option while installing python!

Verify install: (open cmd prompt and enter)

python --version

#See_screenshot_here

Step Two

Install NumPy

  • Goto the website - Numpy (UO)
  • Download suitable version of numpy (we are using : numpy‑1.15.4+mkl‑cp36‑cp36m‑win32.whl)
  • Save it in folder like desktop or downloads
  • open cmd and type:
  • python C://path/to/pip.exe install C://path/to/downloaded/numpy.whl
  • Example: (all in one line)
  • python C:\python\Scripts\pip.exe install C:\Users\Nikk_PC\Downloads\numpy‑1.15.4+mkl‑cp36‑cp36m‑win32.whl
  • Verify Installation:
    • open python in cmd or IDLE
    • Type in below
    • import numpy as np
    • This should not throw any error

Step Three

Install OpenCV

  • Goto this website
  • download suitable opencv whl file
  • we are using : opencv_python-3.4.3-cp36-cp36m-win_amd64.whl
  • same as previous installation
  • Save it in folder like desktop or downloads
  • open cmd and type:
  • python C://path/to/pip.exe install C://path/to/downloaded/opencv.whl
  • Example: (all in one line)
  • python C:\python\Scripts\pip.exe install C:\Users\Nikk_PC\Downloads\opencv_python-3.4.3-cp36-cp36m-win_amd64.whl
  • Verify Installation:
    • open python in cmd or IDLE
    • Type in below
    • import cv2
    • This should not throw any error

    • Step Four

    Install Dependencies/Tools

  • in cmd prompt : python C:/path/to/pip.exe install {tool name}
